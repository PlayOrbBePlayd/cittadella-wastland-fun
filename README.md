A top-down action-adventure game inspired by classic Zelda, set in post-apocalyptic Italy.

## Story

Three weeks ago, your friend Nicole disappeared in the ruins of Cittadella. She left only a journal with a cryptic message about Villa Contarini. Control Kelly as you explore the wasteland, find clues, and discover the truth about The Fracture.

## How to Play

**Controls:**
- **WASD or Arrow Keys** - Move Kelly
- **SPACE** - Interact with objects and NPCs
- **E** - View inventory
